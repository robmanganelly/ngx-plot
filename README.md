# Ngx plot

This project tries to create an ergonomic adapter to use Plot from Observable in Angular.
It is still in early development, but it is already usable.

## Architecture

There are some opinionated architectural choices in this project, that are worth mentioning:

- This library is not a replacement for Plot, but rather an adapter to use it in Angular.
- The APIs of Plot are not wrapped, and the cases where it is convenient, we offer utils to be used in the components, to be used along with the Plot APIs.
